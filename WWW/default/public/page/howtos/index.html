caddy 添加新站点

 1. caddy   (golang 写的可以 自动https 的 HTTP/2 Web服务器)
 2. php     (以 fastcgi 方式运行，默认绑定 9000 端口)
 3. mysql   (5.7 支持 json 类型)
 4. redis   (现在也是标配了)
 5. apache  (给不习惯用 caddy 的人)
